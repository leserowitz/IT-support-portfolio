# Ticket #005 – Teams microphone not detected

**User:** HR  
**Issue:** User can hear calls but cannot speak.  
**Cause:** Windows privacy settings blocking microphone access.  

**Resolution:**  
1. Enabled microphone access in Windows Privacy settings  
2. Restarted Microsoft Teams  
3. Tested call – microphone working  

**Prevention:** Add quick audio-check guide for new laptops.
