# Ticket #001 – Outlook cannot sync inbox

**User:** Marketing  
**Issue:** Outlook shows “Disconnected” and emails don’t update.  
**Cause:** Cached Outlook profile after password change.  

**Resolution:**  
1. Removed old Outlook profile  
2. Created new profile via Control Panel → Mail  
3. Forced autodiscover and updated credentials  
4. Outlook re-synced successfully  

**Prevention:** Add password-change checklist to onboarding guide.
