# IT Offboarding Checklist

## Before Last Day
- Disable user in AD  
- Block sign-in  
- Reset MFA  
- Revoke tokens  

## Hardware
- Collect laptop, charger, badge  
- Check for damage  
- Backup required data  

## Accounts
- Forward inbox (if allowed)  
- Remove licensing  
- Remove distribution groups  

## Security
- Remove VPN access  
- Disable shared drive access  
- Archive mailbox if needed
