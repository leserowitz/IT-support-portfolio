# Printer Troubleshooting Guide

## Common Issues
- Printer offline  
- Stuck print queue  
- Wrong printer selected  
- Network printer unreachable  

## Solutions

### Printer Offline
1. Verify printer is powered on  
2. Check printer IP address  
3. Ping printer  
4. Update Windows printer port  

### Stuck Print Queue
1. Open Services  
2. Restart Print Spooler  
3. Clear pending jobs
