# Microsoft Outlook – Troubleshooting Guide

## Outlook Not Syncing
1. Check bottom status bar (Connected / Disconnected)  
2. Restart Outlook  
3. Update password in Windows Credential Manager  
4. Test connection using Outlook Connection Status (CTRL + right-click Outlook tray icon)  
5. Recreate Outlook profile if required

## Search Not Working
1. Close Outlook  
2. Rebuild Windows Search Index  
3. Restart PC  

## Outlook Keeps Asking for Password
1. Clear old credentials (Control Panel → Credential Manager)  
2. Check network connectivity  
3. Ensure Modern Authentication is enabled
