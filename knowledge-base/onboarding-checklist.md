# IT Onboarding Checklist

## Hardware Setup
- Install Windows updates  
- Join device to domain  
- Configure BitLocker  
- Install antivirus  

## User Accounts
- Create AD account  
- Assign groups  
- Set mailbox  
- Configure MFA  

## Applications
- Microsoft 365  
- Teams  
- Browser setup  
- Business apps  

## Final Steps
- Test login  
- Test email  
- Connect printer  
- Confirm OneDrive sync
