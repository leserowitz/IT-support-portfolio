# Windows – Basic Troubleshooting Guide

## 1. Restart Checklist
- Restart the computer
- Restart the application
- Check for system updates
- Check network connection

## 2. Slow Performance
**Possible Causes:**  
- Too many startup programs  
- Low disk space  
- Heavy background processes  
- Outdated drivers  

**Resolution Steps:**  
1. Open Task Manager → check CPU/RAM usage  
2. Disable unnecessary startup apps  
3. Free disk space (target: 10–20 GB free)  
4. Run Windows Update  
5. Restart the system  

## 3. Application Not Responding
1. End task in Task Manager  
2. Reopen program  
3. Install pending updates  
4. Run program as administrator  

## 4. User Login Issues
1. Confirm CAPS LOCK  
2. Check if account is locked  
3. Reset password  
4. Verify domain selection
