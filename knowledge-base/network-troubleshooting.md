# Network Troubleshooting Guide

## Quick Checklist
- Is Wi-Fi enabled?
- Are other devices affected?
- Does the issue happen on cable and Wi-Fi?
- Is it only one website or all?

## Step-by-Step Diagnostics
1. Ping gateway  
2. Ping external host (8.8.8.8)  
3. Check DNS resolution  
4. Flush DNS  
5. Renew IP  
6. Restart adapter
