# Network Troubleshooting Flow (Text Version)

1. Internet working on other devices?
   - No → Router/firewall issue
   - Yes → Step 2

2. Wi-Fi or Cable?
   - Switch connection type → test again

3. Check IP settings
   - ipconfig /all (Windows)
   - dhclient/ip a (Linux)

4. Ping:
   - Gateway  
   - 8.8.8.8  
   - google.com  

5. DNS Issue?
   - Flush DNS  
   - Change DNS server

6. Still offline?
   - Restart adapter  
   - Replace cable  
   - Test on another network

7. Escalate if still unresolved
