# Printer Troubleshooting Flow (Text Version)

1. Is the printer ON?
   - No → Turn it on
   - Yes → Step 2

2. Is the printer showing an error?
   - Yes → Fix error (paper, toner, jam)
   - No → Step 3

3. Is the printer reachable from PC?
   - Ping printer IP
     - No response → Check network cable / Wi-Fi
     - Responds → Step 4

4. On PC: Is the correct printer selected?
   - No → Set correct printer as default
   - Yes → Step 5

5. Print queue stuck?
   - Yes → Restart Print Spooler
   - No → Reinstall printer driver

6. Still not working?
   - Escalate to 2nd level support
