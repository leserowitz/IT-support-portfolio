# IT Offboarding Checklist

## 1. Accounts & Access
- Disable AD account
- Block sign-in in Azure
- Revoke sessions & MFA
- Remove licenses
- Remove from groups
- Disable VPN access

## 2. Data & Email
- Backup required data
- Forward mailbox (if policy allows)
- Export mailbox if needed
- Remove OneDrive access

## 3. Equipment
- Collect laptop + charger
- Check condition
- Collect badge, keys, headset
- Remove stickers and labels

## 4. Security
- Change shared passwords if used
- Disable shared drive permissions
- Remove user from distribution lists
