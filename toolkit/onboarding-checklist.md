# IT Onboarding Checklist

## 1. Hardware Setup
- Prepare laptop, charger, and accessories
- Install Windows updates
- Join device to domain
- Encrypt drive (BitLocker)
- Install antivirus/EDR
- Configure VPN access

## 2. Accounts & Permissions
- Create AD account
- Assign groups and licenses
- Set mailbox and Teams
- Set MFA + recovery methods

## 3. Applications
- Microsoft 365
- Teams
- Browser setup (bookmarks)
- Business-specific applications
- Printer setup

## 4. Final Steps
- Test login
- Test email
- Test VPN
- Confirm OneDrive sync
- Confirm user has access to all needed systems
